# Portfolio Site - ymizomasa.com

自身のWeb制作スキルを可視化し、クライアントへ安心感と信頼を届けることを目的としたポートフォリオサイトです。

## 🔗 URL
[https://ymizomasa.com/](https://ymizomasa.com/)

## ⏱ 制作時間
- **デザイン（Figma）**: 20時間
- **フロントエンド（HTML/CSS/JS）**: 15時間
- **WordPress構築**: 3時間

## 📝 概要
ミニマルで清潔感のあるデザインをベースに、適切な余白と心地よいアニメーションで「洗練された印象」を表現しています。単なる紹介サイトではなく、実務を想定した保守性の高いコーディングを徹底しました。

## 🎨 デザインコンセプト：「Simple × Motion」
- **視覚効果**: 余白を活かした構成と柔らかいブルーのグラデーションにより、落ち着いた信頼感を醸成。
- **UX/UI**: スクロールやホバー時のマイクロインタラクションにこだわり、ユーザーに「触れていて心地よい」体験を提供。
- **設計方針**: FigmaにてUI設計・配色・タイポグラフィを厳密に定義し、全デバイスで一貫したレイアウトを整理しています。

## 🛠 使用スキル・技術
### **Core Skills**
`HTML` / `CSS` / `JavaScript` / `GSAP` / `WordPress` / `Figma`

### **Technology Details**
- **HTML / CSS**: セマンティックなマークアップを徹底。共通パーツをコンポーネント化し、再利用性とメンテナンス性を高めています。
- **JavaScript / GSAP**: ローディングアニメーションやタイプライター演出、`Intersection Observer API` を活用したフェードイン等、軽量かつリッチな演出を実装。
- **WordPress (Original Theme)**: 
  - 投稿ページや実績一覧を**カスタム投稿タイプ**で構築。
  - クライアント側で容易に更新・管理ができるよう、操作性に配慮したダッシュボード設計。
- **Figma**: ワイヤーフレームからプロトタイプまで作成。デザインと実装の整合性を1px単位で意識したワークフローを構築。

## 💡 成果と学び
デザインから実装までの全工程を一貫して行うことで、実務におけるワークフローの基盤を確立しました。特に「Figmaでの設計」と「コードによる実装」を往復することで、制作効率の最大化と高いデザイン再現性の両立を実感しています。
